add_operation_issue_asset(key_pair, code, seq_num, record) → {WasmTransactionBuilder}

Wraps around TransactionBuilder to add an asset issuance operation to a transaction builder instance.

While add_basic_issue_asset constructs the blind asset record internally, this function allows an issuer to pass in an externally constructed blind asset record. For complicated transactions (e.g. issue and transfers) the client must have a handle on the issuance record for subsequent operations.
